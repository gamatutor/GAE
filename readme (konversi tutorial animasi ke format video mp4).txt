Instalasi :

1. klik setup.exe, setelah selesai instalasi kemudian :

   a. replace TutorialGenerator.exe di folder C:\gamatutor\Product\Engine Animasi GAE\
      dengan TutorialGenerator.exe yang di download dari https://github.com/gamatutor/gamatutor

   b. download ffmpeg.exe dari https://github.com/gamatutor/gamatutor kemudian taruh file ffmpeg.exe 
      tersebut kedalam folder C:\gamatutor (folder instalasi gamatutor) 




Update gamatutor engine :

1. upgrade fitur perekaman suara
2. penambahan fitur konversi ke video (mp4) - beta version
